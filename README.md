# Box Hunt Game

A simple reaction time game built with HTML, CSS, and JavaScript.  
Click the moving dot ("box") as quickly as you can! Your click times and reaction times are tracked and displayed.

---

## Features

- Start, pause, and reset the game
- Dot moves to a random position at a set interval
- Tracks and displays the last 5 click times and reaction times
- Adjustable reaction interval via input

---

## How to Play

1. **Set the reaction time** (in seconds) using the input box.
2. Click **Start** to begin the game.
3. Click the red dot as soon as it moves.
4. Your click times and reaction times will be displayed in the score area.
5. Click **Reset** to clear the scores and stop the dot.

---

## Files

- `index.html` — Main HTML structure
- `style.css` — Styling for the game area, dot, and score display
- `script.js` — Game logic and interactivity

---

## Usage

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Play and enjoy!
